Trabalho de Juros Simples e Composto

Juros Simples

Juros simples é um acréscimo calculado sobre o valor inicial de um aplicação financeira ou de uma compra feita a crédito, por exemplo. O valor inicial de uma dívida, empréstimo ou investimento é chamado de capital. A esse valor é aplicada uma correção, chamada de taxa de juros, que é expressa em porcentagem.

Juros Composto

Juros compostos são a aplicação de juros sobre juros, isto é, os juros compostos são aplicados ao montante de cada período. Juros compostos são muito utilizados pelo sistema financeiro, pois oferece uma rentabilidade melhor. A taxa de juros é sempre aplicada ao somatório do capital no final de cada mês.
